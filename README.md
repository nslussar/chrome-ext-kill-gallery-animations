# Kill Gallery Animations

Chrome extension that disables fade, slide, and transition animations in image gallery lightboxes.

Supported: Lightbox2, bxSlider, ModuloBox.

## Install

1. Go to `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this folder

## Usage

Click the extension icon on any page to activate. Applies to the current tab only.
